# 児童手当計算ツール【2024年改正対応】所得・年齢別に試算 - 技術仕様書

## 概要

**サービス名**: Child Allowance Calculator
**バージョン**: 1.0.0
**更新日**: 2026-05-27
**URL**: https://appadaycreator.com/child-allowance-calculator/

2024年10月改正版の児童手当を所得・子どもの年齢・人数で試算。所得制限撤廃・高校生延長対応。月額・年額を自動計算します。

- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 子ども1人目の年齢区分をセレクトボックスで選択する
2. 2人目・3人目がいる場合は同様に年齢区分を選択する
3. 4人目以降がいる場合は人数を選択する
4. 「計算する」ボタンをクリックする
5. 月額・年額の合計と内訳を確認する（実際は市区町村で要申請）

## よくある質問（FAQ）

**Q: 児童手当計算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 2024年10月の改正でどう変わりましたか？**

①所得制限が完全撤廃 ②支給対象が高校生（18歳年度末）まで延長 ③第3子以降が月30,000円に増額 ④支払い回数が年6回に増加しました。

**Q: 第1子・第2子・第3子の数え方は？**

基本的に現在養育している子どもの数で数えます。大学生など別居の子も要件を満たせばカウントされる場合があります。詳しくはお住まいの市区町村にご確認ください。

**Q: 計算結果はいつから受け取れますか？**

出生届・転入届後に申請が必要です。申請翌月分から支給されます。出生後15日以内に申請すると生まれた月分も受給できます。

**Q: 所得制限はなくなりましたか？**

はい、2024年10月の改正で所得制限・所得上限制限がともに撤廃されました。どの所得でも同額受給できます。
